Fantasy DM Dashboard

Version 1.0

Created by

Tamás Galambos

Fantasy DM Dashboard is a free, offline campaign management tool created for tabletop role-playing game masters.

Development

Tervezte és fejlesztette Tamás Galambos az OpenAI ChatGPT közreműködésével a tervezés, megvalósítás, tesztelés és finomhangolás során.

The application itself does not use or require an AI service.

Data and privacy

Campaign data is saved locally in JSON files. The Dashboard does not upload campaign data to an external server.

Az opcionális online funkciók csak használatkor lépnek kapcsolatba a szolgáltatójukkal. A hivatalos GitHub Pages-oldalon a GoatCounter anonim oldalmegtekintési statisztikát gyűjt; kampányadatokat nem kap meg, a letöltött offline példány pedig nem tölt be analitikát.

Third-party services

  • GoatCounter — privacy-friendly anonymous pageview analytics on the official GitHub Pages site
  • Open-Meteo Historical Weather API — CC BY 4.0
  • Odysee — optional music streaming
  • Wikimedia Commons — optional audio source
  • Internet Archive — optional audio source
